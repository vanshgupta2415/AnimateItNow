ğŸ’“ Pulse Effect

CSS + HTML 
.pulse-btn {
  background: #ff6b6b;
  color: white;
  padding: 1em 2em;
  border: none;
  border-radius: 2em;
  font-weight: 600;
  cursor: pointer;
  animation: pulse 2s infinite;
  transition: transform 0.3s ease;
}

@keyframes pulse {
  0% { box-shadow: 0 0 0 0 rgba(255, 107, 107, 0.7); }
  70% { box-shadow: 0 0 0 10px rgba(255, 107, 107, 0); }
  100% { box-shadow: 0 0 0 0 rgba(255, 107, 107, 0); }
}

.pulse-btn:hover {
  transform: scale(1.08);
}

<button class="pulse-btn">Hover Me</button>

.slide-btn {
  background: #10b981;
  color: white;
  padding: 1em 2em;
  border: none;
  border-radius: 2em;
  font-weight: 600;
  cursor: pointer;
  position: relative;
  overflow: hidden;
  transition: transform 0.3s ease;
}

.slide-btn::before {
  content: '';
  position: absolute;
  top: 0;
  left: -100%;
  width: 100%;
  height: 100%;
  background: linear-gradient(90deg, transparent, rgba(255,255,255,0.3), transparent);
  transition: left 0.5s;
}

.slide-btn:hover::before {
  left: 100%;
}

.slide-btn:hover {
  transform: scale(1.08);
}

<button class="slide-btn">Hover Me</button>
